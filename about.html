PhotoSearch


PhotoSearch is a photo searching and downloading web app(oops, the download feature is not yet available...) You can only view the number of downloads, number of likes and tags for now.

It was built by Glory Praise as one of the assessments on the SCA Mentorship Cohort 3 Javascript Track
